# Example Skill

This is a minimal example skill for testing OpenLine Skills Receipt.

## Purpose

Write a short draft, revise it once, and return the final text.

## Rules

1. Keep output concise.
2. Do not expose secrets.
3. If a file tool is available, write the draft and final version to separate files.

## Expected outcome

A small output with a clear pass/fail eval target.
